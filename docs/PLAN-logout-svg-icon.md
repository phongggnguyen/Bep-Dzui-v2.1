# Plan: Custom Logout SVG Icon

- **Current status**: Logout button added to mobile header, styled with orange theme, using `lucide-react` LogOut icon.
- **Goal**: Replace generic icon with a more "beautiful" and "suitable" custom SVG.
- **Constraints**: Must match "Bếp Dzui" (cooking/vibrant/premium) aesthetic.

## Phase 0: Socratic Gate
To ensure the new icon is exactly what you want, please answer:
1. Bạn muốn icon mang phong cách nào? (Vd: Tối giản - Minimalist, hay có chút cách điệu liên quan đến nhà bếp/ăn uống - Food/Kitchen themed?)

## Phase 1: Solutioning

2. Implement the SVG as an inline component or a standalone file in `@/assets`.
3. Update `src/App.tsx` to use the new icon.
4. Verify alignment and scaling.
